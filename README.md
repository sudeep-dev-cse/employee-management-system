# 🚀 Employee Management System

A modern Employee Management System built using **React**, **Vite**, **Tailwind CSS**, and **Context API**.

The application provides separate dashboards for Admin and Employees to efficiently manage tasks.

---

# ✨ Features

- 🔐 Admin Login
- 👨‍💼 Employee Login
- ✅ Create New Tasks
- 📋 Task Management
- 🟢 Active Tasks
- 🔴 Failed Tasks
- ✔ Completed Tasks
- 📱 Responsive Design
- ⚡ Fast UI using Vite
- 💾 Local Storage Support

---

# 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- Context API
- JavaScript
- HTML5
- CSS3

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/sudeep-dev-cse/employee-management-system.git
```

Install dependencies

```bash
npm install
```

Run Project

```bash
npm run dev
```
